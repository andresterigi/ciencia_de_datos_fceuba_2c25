1Ejercicio 1

Objetivo: Obtener una tabla que muestre el nombre del vendedor, la sucursal y las ventas totales del semestre (suma de Q1 y Q2), ordenada de mayor a menor por ventas totales.
A)
1. Filtrar vendedores con ventas > 50000
2. Crear variable ventas_totales = ventas_q1 + ventas_q2
3. Seleccionar nombre, sucursal, ventas_totales
4. Ordenar descendente por ventas_totales
B)
1. Seleccionar nombre, sucursal, ventas_q1, ventas_q2
2. Crear variable ventas_totales = ventas_q1 + ventas_q2
3. Seleccionar nombre, sucursal, ventas_totales
4. Ordenar descendente por ventas_totales
C)
1. Crear variable ventas_totales = ventas_q1 + ventas_q2
2. Agrupar por sucursal
3. Calcular suma de ventas_totales
4. Ordenar descendente por suma
D)
1. Seleccionar nombre, sucursal
2. Calcular promedio de ventas_q1 y ventas_q2
3. Ordenar ascendente por promedio
4. Mostrar primeros 5 resultados

2Ejercicio 2

Objetivo: Mostrar una tabla con la región, la cantidad de vendedores y el promedio de años de experiencia por región.
A)
1. Filtrar por categoria_contrato = "Planta"
2. Agrupar por region
3. Contar vendedores y calcular promedio de años_empresa
4. Ordenar por cantidad de vendedores
B)
1. Seleccionar region, años_empresa
2. Agrupar por region
3. Contar vendedores y calcular promedio de años_empresa
4. Crear variable experiencia_categoria según años
C)
1. Agrupar por region
2. Contar vendedores y calcular promedio de años_empresa
3. Filtrar regiones con más de 2 vendedores
4. Ordenar alfabéticamente por region
D)
1. Agrupar por region
2. Contar vendedores y calcular promedio de años_empresa
3. Seleccionar region, cantidad_vendedores, promedio_experiencia
4. Mostrar resultado

3Ejercicio 3

Objetivo: Obtener una tabla que muestre los vendedores con "Alto Rendimiento" (ventas totales > 90000) incluyendo nombre, ventas totales y comisión ganada (ventas totales × comision_base).
A)
1. Crear variable ventas_totales = ventas_q1 + ventas_q2
2. Crear variable comision_ganada = ventas_totales × comision_base
3. Filtrar vendedores con ventas_totales > 90000
4. Seleccionar nombre, ventas_totales, comision_ganada
B)
1. Filtrar vendedores con ventas_q1 > 45000
2. Filtrar vendedores con ventas_q2 > 45000
3. Crear variable comision_ganada = (ventas_q1 + ventas_q2) × comision_base
4. Seleccionar nombre, comision_ganada
C)
1. Agrupar por categoria_contrato
2. Calcular promedio de ventas por categoria
3. Filtrar categorias con promedio > 90000
4. Mostrar vendedores de esas categorias
D)
1. Crear variable ventas_totales = ventas_q1 + ventas_q2
2. Ordenar descendente por ventas_totales
3. Seleccionar primeros 3 vendedores
4. Calcular comision para esos vendedores

4Ejercicio 4

Objetivo: Crear una tabla que compare el rendimiento por tipo de contrato, mostrando la categoría de contrato, cantidad de vendedores, ventas totales promedio y la diferencia porcentual entre Q2 y Q1.
A)
1. Crear variable crecimiento = ((ventas_q2 - ventas_q1) / ventas_q1) × 100
2. Crear variable ventas_totales = ventas_q1 + ventas_q2
3. Agrupar por categoria_contrato
4. Contar vendedores, calcular promedio de ventas_totales y promedio de crecimiento
B)
1. Filtrar por ventas_q2 > ventas_q1
2. Agrupar por categoria_contrato
3. Contar vendedores con crecimiento positivo
4. Calcular ventas totales por categoria
C)
1. Agrupar por categoria_contrato
2. Calcular suma de ventas_q1 y ventas_q2
3. Crear ranking por categoria
4. Mostrar diferencia absoluta entre trimestres
D)
1. Seleccionar categoria_contrato, ventas_q1, ventas_q2
2. Crear variable diferencia = ventas_q2 - ventas_q1
3. Agrupar por categoria_contrato
4. Calcular promedio de diferencia por categoria

5Ejercicio 5

Objetivo: Identificar las sucursales "problemáticas" (con ventas promedio por vendedor menores a 40000 en el semestre) y mostrar sucursal, región, cantidad de vendedores y ventas promedio por vendedor.
A)
1. Crear variable ventas_totales = ventas_q1 + ventas_q2
2. Agrupar por sucursal y region
3. Contar vendedores y calcular promedio de ventas_totales
4. Filtrar sucursales con promedio < 40000
B)
1. Filtrar vendedores con ventas_q1 < 20000 o ventas_q2 < 20000
2. Agrupar por sucursal
3. Contar vendedores filtrados
4. Mostrar sucursales con vendedores problemáticos
C)
1. Agrupar por sucursal
2. Calcular ventas totales por sucursal
3. Dividir entre cantidad de vendedores
4. Filtrar con resultado < 40000
D)
1. Crear variable ventas_totales = ventas_q1 + ventas_q2
2. Filtrar vendedores con ventas_totales < 40000
3. Agrupar por sucursal
4. Contar vendedores y mostrar información de sucursal
